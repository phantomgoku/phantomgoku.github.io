Hosting Content

The way I have my content hosted (and it's good for new people who need a static site with nothing in the background) is via Github Pages, which is free.

As long as you have your content set up as the landing page above dictates, anything hosted at yourname dot github dot com should look good.

Custom URL

I bought my cool url from namecheap.com. For a personal webpage, a url for a year is probably cheaper than coffee for a week.

The next step is to direct your DNS settings in name cheap to your github page. That shouldn't really be difficult, And there's numerous tutorials on line to set it up.

Content and CSS
