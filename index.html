Program Interruption Manager

I built a cool and useful thing for the radio station, using on-hand documentation and some brightly colored buttons. Check it out!

Background

Live Audio streams at WSBF go through a machine that adds a delay. This is so if a mistake on air occurs there is a enough time to prevent the mistake from going out on air. Unfortunately, the only available location for the unit was difficult to reach, especially in the cramped desk space. Additionally, we had users in wheelchairs who were unable to access inital location. The finished product is an extension of the user interface for that delay module that was placed in a sensible position, thanks to an empty expansion panel on the mixing board.

Construction

In the documentation for the Program Interruption Manager was the pinout for the serial port of the unit. With a pins jumped, it would send auxilliary control of the unit to addtional pins. The original diagram had the indicator lights (if a button was pressed for example) separate from the switches.

I wanted to incorporate lights and UI into one unit. As i was working on restoring a vintage arcade machine at the time, I picked up some very cool looking Ultralux Chrome Illuminated Pushbuttons from This arcade supply store.

I chose the specific color buttons you can see here due to them most closely resembling the original panel of the manager.

Finished Product and Potential Expansions

The serial port is also capable of pushing how much time is left on the delay module to a microcontroller. However, at the time I was unsure how to interface with the older protocol as well as a seven segment display. I've recommended an expansion of that caliber to be done as a good project for future Chief Engineers at WSBF-FM Clemson. You can see me hardstyling above the finished product below.
