Forays into an Online Presence

January 1, 2014 by Matt

Having a vehicle to promote yourself through is crucial in todays world, and here's how I got my website up and running.

Hosting Content

The way I have my content hosted (and it's good for new people who need a static site with nothing in the background) is via Github Pages, which is free.

As long as you have your content set up as the landing page above dictates, anything hosted at yourname dot github dot com should look good.

Custom URL

I bought my cool url from namecheap.com. For a personal webpage, a url for a year is probably cheaper than coffee for a week.

The next step is to direct your DNS settings in name cheap to your github page. That shouldn't really be difficult, And there's numerous tutorials on line to set it up.

Content and CSS

I'm using bootstrap for my css solutions at the moment. It's very customizable, free, and I prefer the look of a blog style format for my portfolio.

Other Cool Things

I set up email forwarding! You can reach me at matt at mirrorshades.global. namecheap tells you how to easily forward. Cool stuff!

Finished Product

Well I built a dang website! Check it out at mirrorshades.global , but you're already here so you've probably seen it by now. Wow!
